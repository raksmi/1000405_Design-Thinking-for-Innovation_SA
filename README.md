#  CareFlow AI – Transforming Hospital Waiting Experience

## 📌 Project Overview

CareFlow AI is a user-centered healthcare application designed to transform stressful hospital waiting experiences into a calm, engaging, and emotionally supportive journey.

This project applies **Design Thinking principles** to solve real-world problems using **AI, AR, and VR technologies**.

---

##  Objective

To reduce patient anxiety and uncertainty during hospital waiting by providing:

* Real-time queue updates
* Immersive relaxation experiences
* Emotional and mental wellness support



##  Design Thinking Process

### 1. Empathize

* Conducted survey with 6 participants
* Identified emotions: anxiety, boredom, frustration
* Collected real user insights on waiting experiences

### 2. Define

**Problem Statement:**

> How might we reduce anxiety and uncertainty for patients in hospital waiting areas by creating a calm, engaging, and informative experience?



### 3. Ideate

####  Core Features

* Queue Progress Tracker (real-time updates)
* Distraction Hub (videos, games, content)
* Silent Alerts (notifications for turn)
* Mood-Based Interaction
* Feedback System

####  Innovative Features

* 🤖 AI Emotion Detection (personalized suggestions)
* 🌈 AR Relaxation Mode
* 🎧 VR Immersive Experiences
* 🌍 Language Assistance
* 🎶 Smart Audio Therapy
